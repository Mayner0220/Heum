# Heum
## 혜윰 프로젝트의 메인 인공지능 시스템 개발 repository

### 혜윰은 무슨 시스템인가요?
'혜윰'은 순우리말로 '생각, 셈'이라는 의미로, 저희의 프로젝트를 상징하는 단어입니다.

인공지능 분야 중 NLP(자연어 처리)의 기술을 접목시켜 만든 Writing Assistant입니다.

기본적인 오타 수정부터 문법적 오류를 찾아낼 수 있게끔 개발 중입니다.

가장 메인이 되는 기능은 사용자가 입력한 단어와 유사한 단어들을 추천해주는 시스템입니다.

### 어떻게 개발이 되어가고 있나요?
기본적인 인공지능 모델이 구현된 후, 차례대로 서버를 구현하여 Google Extends로 사용이 가능하게 구현할 예정입니다!

많은 응원 부탁드립니다! :)
